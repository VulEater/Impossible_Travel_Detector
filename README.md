# Impossible Travel Detector

## Overview
A Python-based cybersecurity detection system that identifies suspicious login activity based on unrealistic geographic movement between authentication events.

## Features
- Impossible travel detection
- GeoIP enrichment
- Risk scoring
- VPN/hosting provider detection
- MITRE ATT&CK mapping
- Flask dashboard
- SQLite storage
- JSON/CSV exports

## Technologies Used
- Python
- Flask
- SQLite
- Pandas
- Geopy

## MITRE ATT&CK
- T1078 - Valid Accounts
